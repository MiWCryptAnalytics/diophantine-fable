# Theorem target A1 — single-exponential decision below genus 1

Draft v1 · 2026-07-20 · status: **A1-v1 proved modulo classical correctness
facts** (each machine-tested in this repo; citations pending the verification
workflow) · A1-v2 scoped below as an explicit gap ladder.

## Statement (v1 — provable with today's machinery)

> **Theorem A1-v1.** There is an algorithm that, given f ∈ ℤ[u,v] of size
> s = s(f) (Smale's dense convention, implemented in `smale5.poly.size`),
> decides whether f(x,y) = 0 has a solution in ℤ² in time 2^{O(s)},
> **provided** every ℚ-irreducible component of f either (i) has total
> degree ≤ 2, (ii) depends on a single variable, or (iii) has degree 1 in
> one of the variables (graphs of rational functions, any total degree).
>
> (Sparse and dense size conventions are polynomially related — the sparse
> size is ≤ 3× the dense size, and the dense is ≤ poly of the sparse — so
> membership in Smale's class 2^{s^c} does not depend on the convention;
> 2^{O(s)} under the dense measure is the stronger reading.)

The algorithm is `smale5.solvers.pipeline`; the caps confessed in code are
the constants of the 2^{O(s)} bound.

**Priority and budget (post citation pass, 2026-07-20).** The degree-≤2
content is essentially **Lagarias's theorem** (FOCS 1979 / arXiv:math/0611209):
binary quadratics over ℤ are decidable in time 2^{O(L)}, and solvability is
in NP via succinct certificates. A1-v1's value is the executable uniform
pipeline and the LMM/orbit machinery, not priority. Note also that Smale's
actual budget is 2^{s^c} (exponential time, dense size measure) — our
2^{O(s)} accounting is *stronger* than the problem demands, which is
comfortable slack, and the dense measure means high degree buys extra budget
(s ≥ ~(d+1)(d+2)/2). The accounting discipline still matters: see the log
entry "the Nagell trap" for where a wrong exponent hides.

## Cost accounting (proof skeleton)

Every derived integer below (Δ = b²−4ac, β = 4ae−2bd, γ = 4ag₀−d², N, W,
M = |2aΔ|) is a sum of O(1) products of input coefficients: bit-size O(s).

| stratum | algorithm | dominant cost | bound |
|---|---|---|---|
| factor over ℚ | LLL-based | poly(s) | poly(s) |
| univariate components | rational root theorem | integer factorization by trial division ≤ 2^{s/2} | 2^{O(s)} |
| linear | extended gcd | poly(s) | poly(s) |
| deg 2, Δ < 0 | ellipse window scan | window width ≤ 2√disc/(−Δ) + O(1) | 2^{O(s)} |
| deg 2, Δ = 0 | parabola residue classes | M = \|2aβ\| classes | 2^{O(s)} |
| deg 2, Δ = k² | divisor pairs of W | trial-divide \|W\| ≤ 2^{O(s)} | 2^{O(s)} |
| deg 2, Δ > 0 nonsquare | CF unit + LMM reps + orbit walk | see (a)–(c) | 2^{O(s)} |
| deg 1 in one variable | pseudo-division + crossover window (Runge in miniature) | window ≤ 2 + Σ\|coeffs\| | 2^{O(s)} |

(a) **Fundamental unit.** CF of √Δ has period ℓ(Δ) = O(√Δ · log Δ) =
2^{O(s)} steps *(classical; citation pending)*; convergent bit-growth is
O(ℓ · log Δ) = 2^{O(s)}. Note log t itself is 2^{O(s)} bits — storing the
unit is exponential, which is fine; *depending on it in a scan bound is not*
(the Nagell trap).

(b) **LMM class representatives.** Square divisors f² | N by trial division
(2^{O(s)}); square roots of Δ mod |m| via factoring |m| (2^{O(s)}), at most
2^{O(s)} roots; each PQa run enters its reduced cycle and closes within
O(ℓ) = 2^{O(s)} steps.

(c) **Orbit walk.** σ reduces mod M to an element of GL₂(ℤ/M); cycle length
≤ |GL₂(ℤ/M)| < M⁴ = 2^{O(s)}; ≤ 4·(#reps) seeds. Total 2^{O(s)}. ∎

## Correctness pillars

1. Bézout for the linear stratum (tested).
2. The completing-square identity 4a·f = (2ax+by+d)² − Δy² + βy + γ
   (asserted symbolically in the split-hyperbola path; exercised everywhere).
3. **LMM completeness** — every solution class of x² − Δy² = N contains a
   representative produced by some (f, z) PQa run *(Matthews / Mollin /
   Robertson; citation pass running)*. Cross-validated on 400+ instances
   against brute force and against the Nagell scan.
4. **Orbit-walk completeness** (self-contained): solutions fall into
   finitely many ⟨σ⟩ × {±1}-orbits of the representatives; σ mod M has
   finite order, so the forward walk from each seeded sign-variant is a pure
   cycle visiting every residue pattern of its orbit; acceptance depends
   only on residues mod M. Hit ⇒ constructive YES; all cycles clean ⇒ NO.

## The gap ladder (between A1-v1 and Smale #5)

- **G1 — engineering + classical algebra.** Genus-0 components with ≤ 2
  places at infinity and degree ≥ 3 (these are singular curves). *First
  slice landed (2026-07-20): degree-1-in-one-variable components — all
  rational-function graphs — via pseudo-division + crossover window
  (`smale5/solvers/graph.py`), now clause (iii) of the theorem.* Remaining:
  general parametrization (Sendra–Winkler), reducing integral points to
  Pell-type conditions on the parameter; needs height control. Extends
  A1-v1 toward "all components genus 0 with ≤ 2 places".
- **G2 — the first real frontier, sharper than we thought.** Genus-0
  components with ≥ 3 places at infinity: integral points are finite
  (Siegel); classically reduced to Thue/unit equations; effective via Baker.
  The citation pass *settled our hoped-for shortcut negatively*: the best
  known Thue bounds (Bugeaud–Győry 1996, Győry–Yu 2006) give
  log max(|x|,|y|) polynomial in the coefficient **magnitude** H — i.e.
  exponential in the bit-size log H — so enumeration to the bound is doubly
  exponential and **even fixed-degree Thue is not known decidable in
  2^{s^c}**. The open route is Pell-style decision-without-exhibition:
  compact (power-product) representations of units in degree-d fields, the
  analogue of our LMM/orbit walk one rung up. Sharpest open question this
  program has produced, upgraded:
  > Are Thue equations of fixed degree 3 decidable in time 2^{s^c}?
  > (A fortiori: {f : every component has genus 0}, uniformly in deg f.)
- **G3 — the wall.** Positive-genus components carry finitely many sporadic
  integral points; bounding them is effective Siegel. Single-exponential
  effective Siegel for plane curves ⇒ Smale #5 positive. The converse is
  NOT claimed: deciding ≠ exhibiting, and the Pell layer is the standing
  proof of that distinction.
